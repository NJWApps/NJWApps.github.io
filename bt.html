Arduino Bluetooth Serial Port

Application to control an Arduino via a bluetooth serial connection
For more info on this project goto https://anurobot.tech

Download

Download Latest version here.
Note: To run the installer without admin privileges rename it to "Arduino Bluetooth.exe"
All Files and code is avaliable on github.

How to Use

  1. Run Installer
  2. Run Arduino Bluetooth Serial
  3. Select the COM Port of the Bluetooth Serial Connection from the dropdown list
  4. Click "Connect"
  5. Either type commands into the textbox or press a preconfigured Hotkey
